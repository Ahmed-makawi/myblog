# A full-stack blog web app built with Django, supports user auth, profile management, post creation, and dark mode.

A fully functional blog website built with Django and styled with Tailwind CSS. Users can register, log in, create and edit posts with image uploads, and manage their profile including a profile picture. Features session-based authentication, responsive design, and a dark mode toggle.
